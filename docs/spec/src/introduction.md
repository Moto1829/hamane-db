# はじめに

**hamane-db** は Rust 製の組み込み型ベクトルデータベースエンジンです。
「ベクトル版 SQLite」のように、アプリケーションのプロセス内でライブラリとして
動作し、近似最近傍探索 (ANN) とメタデータフィルタ付き検索を提供します。

## 特徴

- **組み込み型**: サーバ不要。`Database::open(path)` するだけで使える。
  永続化なしの `Database::in_memory()` も同じ API で利用可能
- **クラッシュ一貫性**: WAL (Write-Ahead Log) + 不変セグメント + manifest による
  永続化。プロセスがどの時点で落ちても、確認応答済みの書き込みは失われない
- **HNSW インデックス**: セグメントごとに構築される近似最近傍グラフ。
  書き込み直後のデータ (memtable) は正確検索でカバーされるため、
  挿入したレコードは即座に検索可能
- **メタデータフィルタ**: 等価・範囲・論理結合のフィルタを検索に適用。
  選択率に応じて実行戦略 (pre-filter / post-filter) を自動選択
- **SIMD 距離計算**: NEON (aarch64) / AVX2+FMA (x86_64) の距離カーネル
- **少ない依存**: コアは `thiserror` / `crc32c` / `memmap2` / `rand` のみ

## アーキテクチャ概観

```text
                ┌─────────────────────────────┐
                │      公開 API (hamane)       │
                │  Database / Collection       │
                └──────────────┬──────────────┘
                               │
        ┌──────────────────────┼──────────────────────┐
   ┌────▼─────┐        ┌───────▼────────┐      ┌──────▼──────┐
   │ 検索実行  │        │  書き込みパス    │      │  カタログ    │
   │ (マージ + │        │ (WAL→memtable) │      │ (manifest)  │
   │ プランナ) │        └───────┬────────┘      └─────────────┘
   └────┬─────┘                │
   ┌────▼────────────────────── ▼───────┐
   │        インデックス層 (hamane-index) │
   │   Flat (正確) / HNSW (近似)         │
   └────┬───────────────────────────────┘
   ┌────▼───────────────────────────────┐
   │      ストレージ層 (hamane-storage)   │
   │  WAL + 不変セグメント (mmap)         │
   └────────────────────────────────────┘
```

書き込みは WAL に永続化してからインメモリの memtable に反映されます。
memtable が閾値を超えると、不変の**セグメント**としてディスクに書き出され、
同時に HNSW インデックスが構築されます。検索は memtable (正確検索) と
各セグメント (HNSW) の結果を新しい順の優先度でマージして返します。

## クレート構成

| クレート | 役割 |
|---|---|
| `hamane` | 公開 API。利用者はこのクレートだけに依存すればよい |
| `hamane-core` | 型定義 (Record / Filter / Metric) と SIMD 距離カーネル |
| `hamane-index` | Flat 検索と HNSW の実装 |
| `hamane-storage` | WAL・セグメント・manifest・コンパクション |
| `hamane-cli` | コマンドラインツール (`hamane` コマンド) |
| `hamane-server` | HTTP サーバ (axum の薄いラッパ) |
| `hamane-py` | Python バインディング (pyo3。workspace 外、maturin でビルド) |
| `hamane-bench` | SIFT1M ベンチマークハーネス |
