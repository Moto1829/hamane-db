# hamane-db
vector-db engine 

## ベクトルDBとは
ベクトルDB（ベクトルデータベース）は、テキストや画像などを数値ベクトルとして保存し、意味的に近いデータを高速に検索するためのデータベースです。  
通常のキーワード一致だけでなく「内容が似ているもの」を探せるため、RAG（検索拡張生成）や類似文書検索、推薦システムなどで利用されます。

## このリポジトリでの実現イメージ
hamane-db は `vector-db engine` として、次の流れでベクトル検索を実現する想定です。

1. テキストや画像などの入力データを埋め込みモデルでベクトル化する  
2. 生成したベクトルとメタデータを保存する  
3. 検索時にクエリを同様にベクトル化する  
4. コサイン類似度などで近傍ベクトルを検索し、上位結果を返す

この仕組みにより、キーワード一致だけでは拾いにくい「意味的に近い情報」を取得できます。
